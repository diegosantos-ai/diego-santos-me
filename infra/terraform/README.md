# OVH Terraform Lab

Laboratório inicial para validar autenticação e uso do provider OVH com Terraform.

## Arquivos
- `provider.tf`: configuração do provider OVH
- `variables.tf`: variáveis do projeto
- `terraform.tfvars`: valores locais sensíveis
- `main.tf`: teste inicial do laboratório

## Comandos
```bash
terraform fmt
terraform init
terraform validate
terraform plan
